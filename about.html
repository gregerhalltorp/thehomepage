Om mig

Trebarnsfar, trebarnsfar, trebarnsfar, make, vän.

Utvecklare, musikälskare, löpare


Aktuellt just nu: November 2014 t.o.m. juni 2015 är jag föräldraledig och bor med min familj utanför den lilla staden Ban Phe i Thailand. Här försöker jag vara med min familj så mycket som möjligt, lära mig lite Thai och få tid till annat som jag inte hinner i vanliga fall. Det går sådär.


Jag bor med min familj i Sollentuna och arbetar i Stockholm.

Det mesta av min fritid går åt till att vara far. Jag försöker också hålla mig i form. Och hitta sätt att få mer gjort.

Om jag får någon tid över brukar jag försöka hitta små projekt som jag kan använda till att lära mig saker och kanske också lösa något problem som jag har. Men det är inte ofta jag får tid över just nu.

Jag jobbar som systemutvecklare/konsult på ett rätt litet företag som jobbar med CRM-lösningar. Det är ett roligt arbete, fritt med mycket eget ansvar och mycket omväxling. Eftersom systemen är webb-baserade krävs det också kontinuerlig förändring i takt med att nya tekniker, ramverk och metoder dyker upp. Det är ett arbete som kräver ständigt förnyelse och en vilja att hela tiden lära nytt. Vilket passar mig som handen i handsken.

Mina första försök till utveckling gjorde jag i BASIC på en IBM XT-klon som min far köpte någon gång på 80-talet. Därefter gick jag vidare till Pascal innan jag svängde över helt när jag började på universitetet och lärde mig C++ på min första programmeringskurs. Vägen har sedan gått via Java till C#/.NET och nu vidare mot allt mer Javascript. I mitt arbete arbetar jag i hela stacken från SQL Server via C#/.NET till HTML/CSS/Javascript på klientsidan.
Räknar man efter har jag genom åren använt mig av ett fantastiskt antal språk. Från assembler på simulerade RISC-maskiner och utveckling direkt i Java bytecode som på kurserna i Datorarkitektur, via C (standard och med utökningar för parallellprorgammering) över C++ och Java till LISP i kursen i Artificiell Intelligens. Ruby, Perl och Python kommer också in på ett hörn, framförallt i kurser där textbearbetning är nödvändig. I exjobbet använde jag t.ex. Python extensivt. På jobbet har jag till och med fått lov att lära mig utveckla i VBA för att ta fram makron som kan köras automatiskt vid generering av dokument från databasdata via mallar.

Tidigare studerade jag i Uppsala. Länge. Nu har jag avslutat civilingenjörsprogrammet i Molekylär Bioteknik med inriktning mot bioinformatik. Jag har också läst en massa andra kurser, främst inom datavetenskap. När jag lyckas avsluta mitt exjobb kommer jag också att kunna lägga M.Sc. C.S. till titlarna.

Under de senare åren när jag pluggade och under några år när jag inte pluggade, jobbade jag som personlig assistent.

Under tiden i Uppsala engagerade jag mig en del i nationens (Kalmar nations) bostadsstiftelse. Flytten till Sollentuna gjorde det dock omöjligt att fortsätta det engagemanget.
